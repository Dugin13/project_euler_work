# Project Euler 001

A C# console application for solving Project Euler problems.

## Getting Started

### Prerequisites
- .NET 8.0 SDK or later

### Building the Project
```bash
dotnet build
```

### Running the Project
```bash
dotnet run
```

### VS Code Tasks
This project includes the following VS Code tasks:
- **Build**: Press `Ctrl+Shift+P` and run "Tasks: Run Task" → "build"
- **Run**: Press `Ctrl+Shift+P` and run "Tasks: Run Task" → "run"

### Debugging
To debug the application:
1. Open `Program.cs`
2. Set breakpoints as needed
3. Press `F5` to start debugging

## Project Structure
- `Program.cs` - Main application entry point
- `projecteuler001.csproj` - Project file with dependencies and configuration
- `.vscode/tasks.json` - VS Code tasks for building and running

## Next Steps
Start implementing solutions to Project Euler problems in the `Main` method or create separate classes for each problem.
