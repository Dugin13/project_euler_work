# Project Euler 002

A C# console application for solving Project Euler problem 002.

## Project Structure

- `Program.cs` - Main application entry point
- `projecteuler002.csproj` - Project configuration file
- `.vscode/tasks.json` - VS Code build and run tasks

## Getting Started

### Prerequisites

- .NET 10.0 or later
- Visual Studio Code with C# extensions

### Building the Project

You can build the project using:

```bash
dotnet build
```

Or use VS Code's build task (`Ctrl+Shift+P` → "Tasks: Run Task" → "build")

### Running the Project

You can run the project using:

```bash
dotnet run
```

Or use VS Code's run task (`Ctrl+Shift+P` → "Tasks: Run Task" → "run")

### Development

This project is set up with:
- C# Language Support extension
- C# Dev Kit extension
- VS Code tasks for building and running
- Proper project structure for .NET console applications

## About Project Euler

Project Euler is a series of challenging mathematical/computer programming problems that require more than just mathematical insights to solve.
