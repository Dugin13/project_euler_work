# Project Euler 003

A C# console application project for solving Project Euler problem 003.

## Project Structure

- `Program.cs` - Main application entry point
- `projecteuler003.csproj` - Project configuration file
- `.vscode/tasks.json` - VS Code build and run tasks

## Building and Running

### Using VS Code Tasks
- **Build**: Press `Ctrl+Shift+P`, type "Tasks: Run Task", select "build"
- **Run**: Press `Ctrl+Shift+P`, type "Tasks: Run Task", select "run"

### Using Command Line
```bash
# Build the project
dotnet build .

# Run the project
dotnet run .
```

## Requirements

- .NET 10.0 or later
- Visual Studio Code (recommended)

## Getting Started

1. Open the project in Visual Studio Code
2. Use the build task to compile the project
3. Use the run task to execute the application
4. Modify `Program.cs` to implement your Project Euler solution

## Project Euler Problem 003

The largest prime factor of the number 13195 is 29.

What is the largest prime factor of the number 600851475143?
