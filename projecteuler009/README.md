# projecteuler009

This is a C# console application scaffolded for solving Project Euler problem 9.

## How to build

Run the following command in the project root:

```
dotnet build
```

## How to run

Run the following command in the project root:

```
dotnet run
```

## Project structure
- `projecteuler009.csproj`: Project file
- `Program.cs`: Main application entry point
