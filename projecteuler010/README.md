# projecteuler010

This is a C# console application scaffolded using the .NET CLI.

## How to build

Run the following command in the project root:

```
dotnet build .
```

## How to run

Run the following command in the project root:

```
dotnet run .
```

## Project Structure
- `projecteuler010.csproj`: Project file
- `Program.cs`: Main entry point
- `.github/copilot-instructions.md`: Workspace instructions

## Notes
- This project uses .NET 10.0 (update as needed for your environment).
- For custom requirements, update `Program.cs` and other files as needed.
