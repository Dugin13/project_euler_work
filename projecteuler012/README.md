# projecteuler012

This is a C# console project scaffolded for solving Project Euler problem 12.

## How to run

1. Make sure you have the .NET SDK installed.
2. Open a terminal in this directory.
3. Run:
   ```powershell
   dotnet run
   ```

## Project Structure
- `Program.cs`: Main entry point for the console application.
- `projecteuler012.csproj`: Project file.
