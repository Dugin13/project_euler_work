# Project Euler 022

This is a C# console application for solving Project Euler problem 22.

## Getting Started

### Prerequisites
- .NET SDK installed

### Build and Run
1. Open a terminal in the project directory.
2. Build the project:
   ```powershell
   dotnet build
   ```
3. Run the project:
   ```powershell
   dotnet run
   ```

## Project Structure
- `Program.cs`: Main entry point for the application.
- `projecteuler022.csproj`: Project configuration file.
