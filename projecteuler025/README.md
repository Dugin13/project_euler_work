# projecteuler025

This is a C# console application scaffolded for solving Project Euler problem 25.

## How to build and run

1. **Build the project:**
   ```powershell
   dotnet build .
   ```
2. **Run the project:**
   ```powershell
   dotnet run .
   ```

## Project Structure
- `Program.cs`: Main entry point for the application.
- `projecteuler025.csproj`: Project configuration file.

## Requirements
- .NET 6.0 or newer (recommended .NET 10.0)
