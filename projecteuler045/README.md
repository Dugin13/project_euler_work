# Project Euler 045

This is a C# console application for solving Project Euler problem 45.

## Getting Started

This project is intended to be a simple C# console app. You can build and run it using the .NET CLI or Visual Studio Code.

## Requirements
- .NET SDK (6.0 or later recommended)

## Build and Run
1. Open a terminal in the project root.
2. Run `dotnet build .` to build the project.
3. Run `dotnet run .` to execute the program.

## Customization
Add your solution code to `Program.cs`.

---

For more information on Project Euler problems, visit [Project Euler](https://projecteuler.net/problem=45).
