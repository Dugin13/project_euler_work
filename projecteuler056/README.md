# projecteuler056

This is a C# console application scaffolded for solving Project Euler problem 56.

## How to Run

1. Make sure you have the .NET SDK installed.
2. Open a terminal in this directory.
3. Run the following command:
   
   ```powershell
   dotnet run
   ```

## Project Structure
- `Program.cs`: Main entry point for the application.
- `projecteuler056.csproj`: Project configuration file.

## Customization
Add your solution code for Project Euler problems in `Program.cs` or additional files as needed.
