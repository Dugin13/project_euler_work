# projecteuler061

This is a C# console application for solving Project Euler problem 61.

## Getting Started

- Build the project with `dotnet build .`
- Run the project with `dotnet run .`

## Project Structure
- `Program.cs`: Main entry point for the application.
- `.github/copilot-instructions.md`: Copilot instructions and project checklist.

## Requirements
- .NET 10.0 SDK or later
